\section{Result Visualization and Post-Processing}

Post-processing is provided through Tkinter result windows and matplotlib plots. Plotting functions consume the model and result objects; they do not perform solver math. This separation keeps the visual layer replaceable and makes plotted data traceable to numerical result containers.
